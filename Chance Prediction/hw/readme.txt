Run program follow these steps:
1. install packages
	python python -m pip instal requirements.txt
2. run program
	python main.py

Each card will be predicted whether it will be drawn for the next day(or the same day if some lottery has already been made) or not. 
The training and testing accuracy will be shown after ("--------training random forest-----------") print out start from 7 to A
The accuracy will be printed out at the end of the program
Result will be generated as sesult.xlsx
